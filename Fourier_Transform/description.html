Fourier Transform

The Fourier Transform is an operation that transforms a function \(f(t)\) from the time domain into a function \(F(\omega)\), where \(\omega\) describes the frequency domain. Linear operations performed in one domain (time or frequency) have corresponding operations in the other domain, which are sometimes easier to perform.

The drop down menu allows to choose from six different input functions \(f(t)\):

  • rectangular function \(rect(t)=\begin{cases}1&\text{for }-0.5\leq t\leq0.5\\0&\text{otherwise}\end{cases}\)
  • sinc function \(sinc(t)=\frac{sin(t)}{t}\)
  • dirac delta function \(Î´(t)=\begin{cases}\infty&\text{for }t=0\\0&\text{otherwise}\end{cases}\)
  • constant function \(f(t)=c=1\)
  • sine function \(sin(t)\)
  • cosine function \(cos(t)\)

The plots show the input function \(f(t)\) and the corresponding Fourier Transform. The input function \(f(t)\) is defined as the sum of an even function \(f_e(t)=f_e(-t)\) and an odd function \(f_o(t)=-f_o(-t)\). The Fourier Transform of a real and even function is also real and even. The Fourier Transform of a real and odd function is an imaginary and odd function.

Additionally, a second drop down menu allows the modification of the input functions. With this it is possible to show the basic theorems of the Fourier Transform.
