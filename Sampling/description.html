Discrete Fourier Transform

In comparison to the continous Fourier Transform the Discrete Fourier Transform works with discrete data. Instead of a continuous function a limited amount of samples is transformed from the original into the transformed domain.

In this application you can choose a periodic function in original domain x(t), a sampling interval T0 and the number of samples to be used. The choosen function then is discretized and Fouriertransformed from the time (t [s]) to the frequency (f [1/s]) domain. As a result you get the visualization of the discretized function in the original domain (x(t)) and in the Fourier domain (X(f)).

It is important to choose suitable values for the sampling interval and the number of samples in order to get a correct result for X(f). Otherwise Leakage or Aliasing could occur and distort the result. For the two predefinded functions an analytical solution is given in order to check whether distorting effects occur. Furthermore you can check the sampling- and Nyquist frequency for the choosen parameter configuration.
