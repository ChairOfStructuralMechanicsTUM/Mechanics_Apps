Wave Propagation

This app visualizes the wave propagation in a two-dimensional, homogeneous, isotropic, linear-elastic half-space with a vertically pulsating harmonic load. It visualizes the total movement of particles in the soil and the displacement components of the potentials.

The interactivity parameters are

  • soil parameters: damping ratio for material damping $D$, Poisson's ratio $\nu$
  • load parameters: ecitation frequency $\Omega$, width $b$, amplitude $p_{0}$
  • plot area in $z$-direction
  • position of particle for which the potentials are displayed

The wave equation is given by Lam√©: $$Gu^{i}\vert^{j}_{j}+(\lambda+G)u^{j}\vert^{i}_{j}-\rho\ddot{u}^{i} =0 \ \ \ \text{mit}\ i,j = x,y,z$$ The load is defined by the following equation: $$P(x,t) = p_{1}(x) \cdot p_{2}(t) = \begin{cases} p_{0}\cos(\Omega t) & -b \leq x \leq b\\ 0 & x < -b \vee x > b \end{cases}$$ If there is no external stress $p_{0}=0$, only Rayleigh waves occur as result of the harmonic solution of the wave equation. The total shift is made up of the following potentials: $$u = \frac{\partial \Phi}{\partial x}-\frac{\partial \Psi_{y}}{\partial z}$$ $$w = \frac{\partial \Phi}{\partial z}+\frac{\partial \Psi_{y}}{\partial x}$$

Both plots are scaled for a clearer visualization. The scaling factor is $\cdot (10^{6}|\Delta|\Omega)$ for $p_{0}=1$ and $\cdot (10^{2}|\Delta|\frac{1}{\Omega})$ for $p_{0}=0$.
The variable $\Delta$ corresponds to the respective distance between the particles for $u$ in $x$-direction and for $w$ in $z$-direction.
