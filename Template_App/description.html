Template

This app provides a template for the fast start of developing a new app.
It can also be used as a reference resource for specific models when enhancing existing apps.

Directories

main.py - mandatory

The main file where the magic of the app happens.
All other files will find together here, for example the description, images or helper functions.
Most importantly - apart from all the imports - the final page layout is set up in main.

description.html - mandatory

An html-file describing the app. Must be loaded in main.py.
Multiple html-files for descriptions before, after or between certain plots/tools can also be used.
However the main description at the start of every app must always be provided.

constants.py

All constants that are used throughout the app should be defined here.
Constants could be repeatedly used strings, initial slider settings, standard width and height for your widgets, image links, etc.
It is highly encouraged to include such a separate file for constants since it facilitates possible future adjustments without having to look through the rest of the code.

/static/images

All images used in the app must be placed and loaded from here.
