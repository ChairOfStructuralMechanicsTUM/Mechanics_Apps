Wavelet Transform

This app finds the wavelet transform of: Heaviside functions, Rectangular functions, Dirac functions, Trigonometric functions and user defined functions using different two different wavelet functions.

The user is required to input the points in time where the suddun jump happens in a heaviside functions, rectangular function and dirac function along with the desired amplitude. On the other hand, user is asked to input the frequency of the trigonometric function. Please note that user defined functions can be of type:
  • Polynomials
  • Trigonometric functions ( Combinations of sin and cos )
  • Exponential functions

    • In addition, the scaling parameter "a" effecting the center-frequency and the bandwidth can be manipulated from small values (good resolution in time domain) to high values (good resolution in frequency domain).Similarly, the shifting parameter "b" can be varied resulting in a time-shift of the wavelet. For a fixed value of the parameter "a" the transformation is interpreted as a convolution.

    Finally, the user has the ability to define the desired resolution (number of computations per axis). Please note that high resolution values would lead to higher computational need and hence longer time.
