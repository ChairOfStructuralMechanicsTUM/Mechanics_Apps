Wavelet Transform

This app finds the wavelet transform of: Heaviside functions, Rectangular functions, Dirac functions and user defined functions using different wavelet functions.

Scaling parameter "a" effecting the center-frequency and the bandwidth can be manipulated from small values (good resolution in time domain) to high values (good resolution in frequency domain).

Similarly, the shifting parameter "b" can be varied resulting in a time-shift of the wavelet. For a fixed value of the parameter "a" the transformation is interpreted as a convolution.
