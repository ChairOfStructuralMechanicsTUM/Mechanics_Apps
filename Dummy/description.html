Dummy App

This dummy app will show you how to program constantly recurring Bokeh models which can be found in most of the apps of this project.
For a general overview of Bokeh, please consult the Wiki.

Inclination Example

First of all, let's start with a slider. Sliders can be used to dynamically change variables, and its effect is usually found in (several) plots.
In this example, we will change the inclination of a line. All you need is

  • a slider changing the angle $\alpha$
  • a callback function which defines the new coordinates to plot based on $\alpha$
  • a figure to show the output

To show the updated data and omit the old one, Bokeh uses a special type called ColumnDataSource.
A detailed description of the underlying functions and programmatic structure can be found in the code itself.
