You might have noticed a bit of LaTeX in the descriptions. This is rendered due to a special package which is important in the main file. However, currently this only runs up to version 1.0.2 of bokeh. So keep in mind to possibly downgrade the bokeh module if it keeps throwing errors regarding LaTeX.
See the Wiki for more details.

Last but not least one of the most important parts to actually receive output and being able to look at the figures in your favorite browser lies in the command curdoc().add_root(), where you can define the position your objects (figures, sliders, buttons, etc.) by using rows and columns notation.
Without this last step - opening the window the outside world - all your programming and ideas would just stay imaginary.
