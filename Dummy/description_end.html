LaTeX/KaTeX and last steps

You might have noticed a bit of LaTeX in the descriptions. This is rendered due to a special package (KaTeX) which is imported in the main file. It is not only possible to include LaTeX in the description text but also in the title of sliders, or legends and labels of plots.
When updating the Bokeh version, you might have to adapt the custom LaTeX implementation in the shared folder. See the Wiki for more details.

Last but not least one of the most important parts to actually receive output and being able to look at the figures in your favorite browser lies in the command curdoc().add_root(), where you can define the position of your objects (figures, sliders, buttons, etc.) by using rows and columns notation.
Without this last step - opening the window to the outside world - all your programming and ideas would just stay imaginary.
