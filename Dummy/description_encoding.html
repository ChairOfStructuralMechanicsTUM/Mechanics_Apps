Encoding

For some apps it is possible to switch between an English and German version. In order to display umlauts properly, we need the utf-8 encoding.

To provide this feature, you need a file strings.json in the static/ folder where also the images are located.
See the Template App and for example Vector_addition or Seesaw.

The important part to know is, to provide an extra parameter for the open function, loading html files into strings.json or loading strings.json into your python files.
open('Seesaw/description_de.html', encoding='utf-8').read()
