Hide Example

Sometimes hiding objects for specific settings becomes handy. Both the user experience improves and the app is less crowded.
Bokeh provides a simple way to achieve this. In old bokeh versions, one had to manipulate the objects via JavaScript and CSS. Now it is enough to set the visible attribute accordingly.

Of course we could also add some functionality to the button via callback functions like in the example above.
For the purpose of providing a short overview we keep it this way without functionality. Nevertheless, feel free to try to expand or interconnect the examples.
A good "real world" reference of the apps can be considered to be SDOF, since it provides a structure using classes, many sliders, buttons and animations.
