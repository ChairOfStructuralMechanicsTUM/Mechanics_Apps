For this more sophisticated example of hiding a button, we need to dive a bit into CSS and JavaScript, since Bokeh does not offer yet a simple hide functionality.
We need:

  • a (JavaScript) callback function
  • a CSS style file located in /templates/styles.css
  • an html index file located in /templates/index.html

All or the most bokeh models provide a css_classes attribute.
Unfortunately, simply altering the css_classes attribute via hide_button.css_classes has no effect. That is why we need explicit JavaScript code which changes the underlying CSS names of the object in question.
UPDATE: It seems that now it is in fact possible to change the CSS classes without having to use a JavaScript function. A normal callback function programmed in python like in the examples above will do.

Choose any tag, here "hidden" which should be present in the css_classes. The same tag is used in styles.css, so it is very important to not change this tag!

Of course we could also add some functionality to the button via callback functions like in the example above.
For the purpose of providing a short overview we leave it this way without functionality. Nevertheless, feel free to try to expand or interconnect the examples.
