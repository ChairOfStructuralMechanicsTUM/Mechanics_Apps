Loading Example

For very long and exhaustive computations it is beneficial to inform the user about that. Most commonly this is done through a loading symbol.
CSS codes for such loading animations can be found for example on https://loading.io/css/.

We need:

  • an html file including the CSS style definition
  • a Div setting this loading symbol
  • curdoc().add_next_tick_callback

By manipulating the layout one can replace for example a figure with the loading symbol. One could also just provide a dedicated space for this sign and set it to visible only when needed.
We go for the second approach here. The other way can be seen in the Wavelet App.

Since Bokeh currently synchronizes only at the end of callback functions, we need to provide an extra work function where we put the heavy computation.
This function is then called via a curdoc().add_next_tick_callback in order to end the original callback which was opened by a button for example.
