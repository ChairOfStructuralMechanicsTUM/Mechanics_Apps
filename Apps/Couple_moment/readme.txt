Moment_2 App

Prototype of a WebApp demonstrating Moment vector via Bokeh for 
the lectures at the Chair of Structural Mechanics, Prof. Müller, TUM (Technische Universität München).

Location of force can be changed and correspoding change in magnitude for same Moment can be plotted.

The app can be run using the command "bokeh serve --show Couple_moment" from the parent directory.
