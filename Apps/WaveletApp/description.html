Wavelet Transform

Wavelets can be used as a mathematical tool in order to describe functions complementary to a Fourier-synthesis. In a Fourier-description, each Fourier-series member has its specific frequency without having a spatial localization, whereas a wavelet-based synthesis provides this spatial localization.

The Wavelet transform assigns a function $f(t)$ of a real variable $t$ to a function $\mathcal W_f (b,\,a)$, where $\mathcal W_f (b,\,a)$ is the wavelet-transform of $f(t)$.

The transformation rule is

$$ W_{f}(a,b) = \mid a \mid^{-\frac{1}{2}} \int_{-\infty}^{\infty} f(t) \: \psi(\frac{t-b}{a}) \: dt,$$

where $\psi\left(t\right) \in L_2(\mathbb{R})$ is the wavelet.

The wavelet-transform $\mathcal W_f (b,\,a)$ contains both, time information and frequency content of $f(t)$. Depending on the value of $a$ those characteristics of $f(t)$ are resolved by $\mathcal W_f (b,\,a)$ . While a small value for $a$ results in a good resolution of the time information and a poor representation of the frequency content, a big value for $a$ leads to a good resolution of the frequency content and thus a bad resolution of the time information.

This app visualizes the wavelet-transform of: Heaviside functions, rectangular functions, Dirac delta functions, trigonometric functions and user defined functions using different two different wavelet functions.

Wavelet 1: $\psi\left(t\right) = t\, e^{-t^2}$

Wavelet 2: $\psi\left(t\right) = e^{-\frac{t^2}{2}} \, \cos(5\,t)$

The interactivity parameters are:

  • Choosing a function $f(t)$ for which the Wavelet transform should be performed. Please note that user defined functions can be of type: polynomials, trigonometric functions, exponential functions and combinations of those function types.
  • Choosing a wavelet (Wavelet 1 or Wavelet 2).
  • Visualize the chosen wavelet for values of a (scaling) and b (shifting).
  • Define the desired resolution of $\mathcal W_f (b,\,a)$. Please note that high resolution values would lead to higher computation times.
